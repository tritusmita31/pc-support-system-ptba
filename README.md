# 💻 PC Support System

Aplikasi berbasis Streamlit untuk monitoring perbaikan perangkat IT (Laptop, PC, Printer).

## ✨ Fitur
- Login Admin & User
- Pengajuan tiket perbaikan
- Tracking status perangkat
- Notifikasi email otomatis
- Dashboard monitoring

## 🚀 Cara Menjalankan
```bash
streamlit run app.py
